✦ ABOUT
What is PDF Suite?
A fully browser-based toolkit for converting, compressing, watermarking, protecting, merging and splitting PDF and image files — zero server, zero uploads, zero compromise on privacy.
100%
CLIENT-SIDE
0
FILES UPLOADED
4
CORE TOOLS
🖼️
Image → PDF
Convert JPG, PNG, WEBP images to PDF. Drag to reorder. Rotate/flip per image. Watermark, password, page numbers, background color.
📄
PDF → Images
Extract every page as PNG or JPEG at up to 216 DPI. Download individually or all as ZIP.
🔀
Merge PDFs
Combine 2+ PDF files in order. Pages preserved at full quality using PDF.js rendering.
✂️
Split PDF
Split a PDF into individual pages or custom ranges. Download all splits as ZIP instantly.

🛠 Tech Stack

jsPDF 2.5.1
PDF.js 3.11
JSZip 3.10.1
Vanilla JS
Canvas API
