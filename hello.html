✦ ABOUT
What is PDF Suite?
A fully browser-based toolkit for converting, compressing, watermarking, protecting and merging PDF and image files — zero server, zero uploads, zero compromise on privacy.
100%
CLIENT-SIDE
0
FILES UPLOADED

CORE TOOLS
🖼️
Image → PDF
Convert JPG, PNG, WEBP, GIF and BMP images into a single PDF. Control page size, orientation, margins, quality and add watermarks or password protection.
📄
PDF → Images
Extract every page of a PDF as a high-resolution PNG or JPEG. Choose resolution (up to 216 DPI) and custom page ranges. Download individually or all at once as ZIP.
🔀
Merge PDFs
Combine two or more PDF files into one seamless document. Pages are preserved in order and merged at 1.5× scale for crisp quality.
🔒
Password Protection
Encrypt your exported PDFs with a password so only authorised recipients can open them. Powered entirely in-browser via jsPDF's encryption API.

🛠 Tech Stack

Built with zero frameworks and zero back-end dependencies. Everything runs in your browser using battle-tested open-source libraries:

jsPDF 2.5.1 — PDF generation
PDF.js 3.11 — PDF rendering
JSZip 3.10.1 — ZIP packaging
Syne + Space Mono — Typography
Vanilla JS — No frameworks
Canvas API — Image processing

📊 What We Collect

Nothing. There is no backend server for this tool. When you convert an image or PDF, the computation runs entirely on your machine. We collect no telemetry, no usage data, no file metadata, and no personal information.

💾 LocalStorage

The conversion history feature stores a brief log of your last 10 conversions in your browser's localStorage. This includes the output filename and some metadata (page count, file size). This data:

  • Is stored only on your device
  • Is never transmitted anywhere
  • Can be cleared at any time via the History tab → "Clear All"
  • Is automatically limited to the 10 most recent entries
